* $x.shift() выяснить (move:): This macro behaves like (put:), but it requires both sides of the into expression to be variables, 
  and deletes the left side after copying its value to the right side. In other words, running (move: $x into $y) will change $y to 
  $x's value and delete $x. Its main intended use is with arrays: (move: $x's 1st into $y) will remove the item at index 0 from the 
  array and set $y to it, in effect behaving like $x.shift() while being more explicit than Javascript and Perl's horridly named stack-manipulation methods.
